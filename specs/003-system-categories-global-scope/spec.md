# Feature Specification: System Categories - Global Scope Refactoring

**Feature Branch**: `003-system-categories-global-scope`  
**Created**: 2026-02-19  
**Status**: Ready for Implementation  
**Type**: Infrastructure/Domain Refactoring  
**Effort**: 5 days | **Risk**: MEDIUM | **Impact**: Breaking (internal only)  

---

## Quick Reference

- **Scope Layers**: Domain + Infrastructure + Application (Minimal Changes)
- **Current State**: System categories are in-memory generated by `CategoryService.GetSystemDefaults(UserId userId)`
- **Target State**: System categories are persisted in database with NULL user_id, available to all users automatically
- **Impact**: Database migration, Repository query logic, Category entity nullable UserId, 3 handler updates
- **Breaking Change**: NO user-facing changes; internal refactoring only
- **Backward Compatibility**: ✅ Zero downtime deployment possible via blue-green strategy

## Critical Problem Statement

### Current Architecture (Feature 2)

**In-memory generation of system categories:**
```csharp
// CategoryService.GetSystemDefaults(UserId userId)
// Called for each user, generates NEW Category instances every time
// System categories are NOT persisted in database
public IReadOnlyList<Category> GetSystemDefaults(UserId userId)
{
    var categories = new List<Category>();
    categories.Add(new Category(...userId...));  // ← per-user, per-call instance
    return categories.AsReadOnly();
}
```

**Issues with Current Design:**
1. System categories are **not persisted** — generated in-memory on every call
2. CategoryService.GetSystemDefaults() requires userId parameter — semantically confusing (why pass user ID for system categories?)
3. Inefficient: Creates 24 new Category objects per user per query (no caching)
4. Inconsistent with architecture: Other domain entities are persisted in database
5. Repository has GetSystemDefaultsAsync(userId) that doesn't match actual usage

### Target Architecture (This Feature)

**Persisted system categories with NULL user_id:**
```sql
-- Persisted once in database:
INSERT INTO categories (id, user_id, name, type, color, icon_name, is_system_default, created_at)
VALUES 
  ('uuid-1', NULL, 'Salary', 'Income', '#27AE60', 'building-dollar', true, NOW()),
  ('uuid-2', NULL, 'Housing', 'Expense', '#E74C3C', 'house', true, NOW()),
  -- ... 22 more (24 total)
ON CONFLICT (user_id, name) DO NOTHING;
```

**Benefits:**
1. ✅ System categories **persisted** once, shared across all users
2. ✅ Explicit NULL semantics: "NULL user_id = system/global category"
3. ✅ Query clean: `WHERE user_id = @userId OR user_id IS NULL`
4. ✅ Consistent architecture: System categories = domain entities = persisted
5. ✅ RLS policies simpler: Direct NULL checks vs magic strings
6. ✅ Scalable: Pattern reusable for future team/shared categories

## Executive Summary (In Scope / Deferred)

### In Scope

1. **Database Migration**
   - Update `categories` table: change `user_id` from non-nullable to nullable
   - Migrate existing `user_id = 'system'` records to `user_id = NULL`
   - Update RLS policies to reflect new logic

2. **Domain Layer Changes**
   - `Category` entity: support `UserId` as nullable (optional property)
   - `CategoryService.GetSystemDefaults()`: return categories with null UserId
   - Repository interface: update signatures if needed (minimal change)

3. **Application Layer Changes**
   - `SupabaseCategoryRepository.GetByUserIdAsync()`: union query (user-specific + system)
   - Handler logic: no changes needed (repository handles the union transparently)

4. **Infrastructure Queries**
   - `GetAllCategoriesQuery`: filter results to remove duplicates (if returned by repository)
   - All existing handlers: no changes (backward compatible)

5. **Tests**
   - Update tests to verify NULL user_id semantics
   - Verify RLS policies work correctly
   - Verify system categories are globally visible

## Architecture & Implementation Details

### Domain Layer

**Target Category Entity (Nullable UserId):**
```csharp
public class Category : AggregateRoot<CategoryId>
{
    public UserId? UserId { get; private set; }  // Nullable: null = system/global
    public CategoryName Name { get; private set; }
    public CategoryType Type { get; private set; }
    public ColorHex Color { get; private set; }
    public string IconName { get; private set; }
    public bool IsSystemDefault { get; private set; }

    // Private constructor accepts nullable UserId
    private Category(
        CategoryId id,
        UserId? userId,
        CategoryName name,
        CategoryType type,
        ColorHex color,
        string iconName,
        bool isSystemDefault)
        : base(id)
    {
        // Domain invariant: NULL user_id requires IsSystemDefault=true
        if (userId == null && !isSystemDefault)
            throw new DomainException("Categories with null UserId must be marked as system defaults.");

        UserId = userId;
        Name = name ?? throw new ArgumentNullException(nameof(name));
        Type = type;
        Color = color ?? throw new ArgumentNullException(nameof(color));
        IconName = iconName ?? throw new ArgumentNullException(nameof(iconName));
        IsSystemDefault = isSystemDefault;
    }

    // Public constructor for user-scoped categories (userId REQUIRED)
    public Category(
        CategoryId id,
        UserId userId,  // Non-nullable for user-scoped
        CategoryName name,
        CategoryType type,
        ColorHex color,
        string iconName)
        : this(
            id,
            userId ?? throw new ArgumentNullException(nameof(userId)),
            name,
            type,
            color,
            iconName,
            isSystemDefault: false)
    {
    }

    // NEW: Factory method for system defaults (no userId parameter)
    public static Category CreateSystemDefault(
        CategoryId id,
        CategoryName name,
        CategoryType type,
        ColorHex color,
        string iconName)
    {
        return new Category(
            id,
            userId: null,  // ← SYSTEM CATEGORIES HAVE NO OWNER
            name,
            type,
            color,
            iconName,
            isSystemDefault: true);
    }

    // Helper methods for semantic clarity
    public bool IsGlobal => UserId is null;
    public bool IsUserScoped => UserId is not null;

    public bool IsOwnedByUser(UserId userId)
    {
        return UserId != null && UserId.Value == userId.Value;
    }

    public bool IsAccessibleToUser(UserId userId)
    {
        return IsSystemDefault || IsOwnedByUser(userId);
    }
}
```

**Category Service Changes:**
```csharp
public class CategoryService
{
    private static IReadOnlyList<Category>? _cachedSystemDefaults;
    private static readonly object _cacheLock = new();

    // GetSystemDefaults now returns cached hardcoded reference (no userId param)
    public IReadOnlyList<Category> GetSystemDefaults()
    {
        if (_cachedSystemDefaults != null)
            return _cachedSystemDefaults;

        lock (_cacheLock)
        {
            if (_cachedSystemDefaults != null)
                return _cachedSystemDefaults;

            var categories = new List<Category>();
            // 24 hardcoded system categories with NULL user_id
            categories.Add(Category.CreateSystemDefault(
                CategoryId.New(),
                CategoryName.Create("Salary"),
                CategoryType.Income,
                ColorHex.Create("#27AE60"),
                "building-dollar"));
            // ... 23 more categories

            _cachedSystemDefaults = categories.AsReadOnly();
            return _cachedSystemDefaults;
        }
    }
}
```

**Repository Implementation Changes:**
```csharp
public class SupabaseCategoryRepository : ICategoryRepository
{
    // GetByUserIdAsync now returns union of user + system categories
    public async Task<IReadOnlyList<Category>> GetByUserIdAsync(UserId userId)
    {
        var response = await _client.From<CategoryRow>()
            .Where(x => x.UserId == userId.Value || x.UserId == null)  // ← UNION query
            .Order("name", Constants.Ordering.Ascending)
            .Get();

        return response.Models.Select(r => r.ToDomain()).ToList().AsReadOnly();
    }

    // NEW: FindByNameAsync checks across all scopes (user + system)
    public async Task<Category?> FindByNameAsync(string name)
    {
        var response = await _client.From<CategoryRow>()
            .Where(x => x.Name == name)  // ← No user_id filter
            .Limit(1)
            .Get();

        return response.Models.FirstOrDefault()?.ToDomain();
    }

    // UPDATED: Remove userId parameter from GetSystemDefaultsAsync
    public async Task<IReadOnlyList<Category>> GetSystemDefaultsAsync()
    {
        var response = await _client.From<CategoryRow>()
            .Where(x => x.UserId == null)  // ← Changed from == userId.Value
            .Where(x => x.IsSystemDefault == true)
            .Get();

        return response.Models.Select(r => r.ToDomain()).ToList().AsReadOnly();
    }

    // Mapping: Handle nullable UserId from database
    private Category ToDomain(CategoryRow row)
    {
        var categoryType = row.Type == "Income" ? CategoryType.Income : CategoryType.Expense;
        UserId? userId = string.IsNullOrEmpty(row.UserId) ? null : new UserId(row.UserId);

        if (row.IsSystemDefault)
        {
            return Category.CreateSystemDefault(
                new CategoryId(Guid.Parse(row.Id)),
                CategoryName.Create(row.Name),
                categoryType,
                ColorHex.Create(row.Color),
                row.IconName);
        }

        return new Category(
            new CategoryId(Guid.Parse(row.Id)),
            userId ?? throw new DomainException($"User category {row.Id} missing user_id"),
            CategoryName.Create(row.Name),
            categoryType,
            ColorHex.Create(row.Color),
            row.IconName);
    }
}
```

**Minimal Handler Changes (Safe null-checking):**
```csharp
// In CreateTransactionCommandHandler, UpdateTransactionCommandHandler, DeleteTransactionCommandHandler:

    throw new UnauthorizedAccessException("You don't own this category");

    throw new UnauthorizedAccessException("You don't have access to this category");
```

**CreateCategoryCommand validation update:**
```csharp
// Validates name against system defaults (hardcoded + database)
var duplicate = await _categoryRepository.FindByNameAsync(request.Name);
if (duplicate != null)
    throw new DomainException($"Category name '{request.Name}' is already in use (system or custom)");

var systemDefaults = _categoryService.GetSystemDefaults();
if (systemDefaults.Any(c => c.Name.Value.Equals(request.Name, StringComparison.OrdinalIgnoreCase)))
    throw new DomainException($"Category name '{request.Name}' is reserved for system defaults.");
```
